# AcuteBoiler
A specialized boilerplate template for quickly developing angular websites using material design.

### Quick Start
    $ git clone https://github.com/SparksD2145/AcuteBoiler.git
    $ cd ./AcuteBoiler
    $ npm install && bower install
    $ node AcuteBoiler
    
### Building
    $ gulp
